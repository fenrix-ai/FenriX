## Bakery Bash SVG UI / icon style (Stardew-valley-esque)

These SVGs are **pixel-art vectors** (rect-per-pixel) so they scale cleanly while keeping a cozy, retro look.

### Conventions
- **Transparent background**
- **Crisp pixels**: `shape-rendering="crispEdges"`
- **Grid**: each pixel is `1x1` in the SVG coordinate system
- **Icon size**: default `32x32` (unless stated otherwise)
- **Outline**: use a dark outline for readability on light/dark UI
- **Shading**: 1–2 shade steps max (simple, readable at small sizes)

### Recommended palette
Based on the Bakery Bash presentation theme:
- `espresso`: `#2C1810`
- `dark-roast`: `#3D2317`
- `cream`: `#FFF8F0`
- `flour`: `#F0E6D8`
- `dough`: `#E8D5BE`
- `caramel`: `#C4873B`
- `golden`: `#E8A849`
- `honey`: `#F5C96A`
- `sage`: `#7A9E7E`
- `berry`: `#B54B6C`

Character outlines already use near-black (`#1c1a1a`). Icons may reuse that for consistency.
